# FMP Vessel Cost Builder

Internal costing tool for fabricating industrial pressure vessels. Users input geometry and material; the app calculates shell optimization, dish ends, and total fabrication cost.

**Stack:** React 18 + TypeScript + Vite | Tailwind CSS + shadcn/ui | Firebase Auth + Firestore | React Hook Form + Zod | Recharts | Vitest + Playwright

**Path alias:** `@/*` → `src/*`
